About

Projects
Optimus

Optimus is an online furniture portal which lets a customer assemble and request for a customized furniture unit. At the end it notifies the customer with details about cost / delivery etc of the unit.

Sudoku Solver

This is a basic Sudoku Solving client that supports only 9x9 sudokus. The logic is implemented in JS. The source supports other squared sudokus and can be used as an external script for any project.
